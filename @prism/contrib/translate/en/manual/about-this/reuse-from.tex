\section{Reuse from...}

Here are the key points to remember when using palettes similar to those offered by projects listed in the section \ref{projects-used}.
%
\begin{enumerate}
	\item
	\thisproj\ uses 'semantic' \verb+CamelCase+ naming. Therefore, palette names such as
	\tdoccodein{text}{berlin}, \tdoccodein{text}{gist_heat} and \tdoccodein{text}{Pinkgrey}
	become
	\tdoccodein{text}{Berlin}, \tdoccodein{text}{GistHeat} and \tdoccodein{text}{PinkGrey}.
	% This implies that palette names from different projects are compared internally using lowercase.
	% Appendix 1 (page \pageref{palette-all-names}) lists all palette names to help you find the correct \verb+CamelCase+ spelling.

    \item \matplotlib\ palettes with a name ending with \verb+_r+ (reversed color order) are not included in \thisproj.
    
    \subimport{../../../common/report/}{renamed-palettes.latex}
    \subimport{../../../common/report/}{ignored-palettes.latex}
\end{enumerate}


% -------------------- %


\begin{tdoccaut}
    Most \thisproj\ implementations add the \verb+pal+ prefix to standardized \verb+CamelCase+ names.
    See the section \ref{products-all}.
\end{tdoccaut}


\begin{tdocnote}
    Most \thisproj\ implementations provide methods to easily obtain reversed palettes, sub-palettes, and color-shifted palettes.
    See the section \ref{products-all}.
\end{tdocnote}
