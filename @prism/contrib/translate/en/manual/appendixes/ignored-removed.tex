The following palettes are excluded as sublists%
\footnote{
A palette is defined as a sublist if it is formed by removing colors while preserving the relative order of the remaining ones.
}
or shifts of existing ones, or due to direct or reversed duplication. Note that exact duplicates -- sharing both name and colors -- are omitted entirely. We identify relation between palettes using the following notations.
%
\begin{multicols}{4}
\begin{itemize}[label=\textbullet\,]
    \item Sublist: \boxed{{\prec}\vphantom{pM}}
    
    \item Shifted: \boxed{{\ll}\vphantom{pM}}

    \item Equality: \boxed{{=}\vphantom{pM}}

    \item Reversal: \boxed{{\rightleftharpoons}\vphantom{pM}}
\end{itemize}
\end{multicols}

In these cases, the rightmost palette is the one retained in \thisproj. For example, this table indicates that \tdoccodein{text}{YlOrRd} from \colorbrewer\ uses a sublist of colors from the \thisproj\ palette \tdoccodein{text}{YlOrRd}.
