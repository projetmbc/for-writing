The following palettes are excluded as subsets of existing ones, or due to direct or reversed duplication. Note that exact duplicates -- sharing both name and colors -- are omitted entirely. We identify related versions using the following notation.
%
\begin{multicols}{3}
\begin{itemize}[label=\textbullet\,]
    \item Subset: \boxed{{\prec}\vphantom{pM}}

    \item Equality: \boxed{{=}\vphantom{pM}}

    \item Reversal: \boxed{{\rightleftharpoons}\vphantom{pM}}
\end{itemize}
\end{multicols}

In these cases, the rightmost palette is the one retained in \thisproj.
