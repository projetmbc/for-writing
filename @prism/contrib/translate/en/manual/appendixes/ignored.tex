The following types of palettes are unkept.
%
\begin{itemize}
    \item \textbf{Exact duplicates:} identical color arrays.

    \item \textbf{Visual identical:} those indistinguishable to the naked eye.%
    \footnote{
        In these cases, the original palette is retained.
        For instance, \matplotlib's  \tdoccodein{text}{Cubehelix} is a specific implementation of the original \cubehelix's \tdoccodein{text}{Classic}.
    }

    \item \textbf{Transformations:} reversals, shifts, or a combination of both.
\end{itemize}
%
We identify relations between palettes using the following notations.
