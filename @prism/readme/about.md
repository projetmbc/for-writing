About @prism
------------

This project provides a collection of discrete color palettes for various programming languages,
enabling the creation and use of color maps derived from these palettes. **For a simple live demonstration, [click here](https://projetmbc.github.io/for-writing/@prism/css/showcase/dark-or-std-mode.html).**



> ***CAUTION.*** *Only discrete palettes are provided. No continuous colormaps are implemented.*
