1. Le cas "Colorblind" (Accessibilité)

C'est l'exemple le plus courant. Une palette peut être à la fois Sequential (pour un dégradé) et Colorblind (parce que les contrastes de luminosité permettent aux personnes daltoniennes de distinguer les étapes).

    Exemple : La célèbre palette Viridis est techniquement Sequential, mais elle est presque toujours étiquetée Colorblind car elle est conçue pour être lisible par tous.

2. Le cas "Divergent" vs "Sequential"

Une palette Divergent est techniquement l'assemblage de deux palettes Sequential qui se rejoignent sur un point neutre (souvent le blanc ou le gris).

    Si vous n'utilisez qu'une moitié de la palette (du neutre vers un extrême), elle devient Sequential.

3. Le cas "Qualitative" vs "Sequential"

Certaines palettes Qualitative (couleurs distinctes) peuvent être perçues comme Sequential si les couleurs suivent une logique de luminosité (du plus clair au plus foncé), même si les teintes changent.

.element {
  background-color: map-get($accents, 1);

@function map-reverse($map) {
  $keys: map-keys($map);
  $reversed-map: ();

  @for $i from length($keys) through 1 {
    $key: nth($keys, $i);
    $reversed-map: map-merge($reversed-map, ($key: map-get($map, $key)));
  }

  return $reversed-map;
}

$accents-reversed: map-reverse($accents);


@function map-shift($map, $steps: 1) {
  $keys: map-keys($map);
  $len: length($keys);
  $shifted-map: ();

  @for $i from 1 through $len {
    // Calcul du nouvel index circulaire
    $new-index: ($i + $steps - 1) % $len + 1;

    $old-key: nth($keys, $i);
    $new-val: map-get($map, nth($keys, $new-index));

    $shifted-map: map-merge($shifted-map, ($old-key: $new-val));
  }

  return $shifted-map;
}

$accents-shifted: map-shift($accents, 1);


@function map-pick($map, $keys-to-keep) {
  $filtered-map: ();

  @each $key in $keys-to-keep {
    @if map-has-key($map, $key) {
      $filtered-map: map-merge($filtered-map, ($key: map-get($map, $key)));
    }
  }

  return $filtered-map;
}

// Utilisation : on ne garde que la 1, la 4 et la 10
$accents-selective: map-pick($accents, 1 4 10);



@function transform-palette-ordered($map, $extract: null, $shift: 0, $reverse: false) {
  $result: $map;

  // 1. EXTRACTION (Pick)
  @if $extract != null {
    $result: map-pick($result, $extract);
  }

  // 2. SHIFT
  @if $shift != 0 {
    $result: map-shift($result, $shift);
  }

  // 3. REVERSE
  @if $reverse == true {
    $result: map-reverse($result);
  }

  return $result;
}

$theme-configs: (

  // Autre exemple : seulement un shift et un reverse
